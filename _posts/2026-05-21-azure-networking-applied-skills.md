I'd like to share another achievement – I passed the [Microsoft Applied Skills: Configure secure access to your workloads using Azure networking](https://learn.microsoft.com/en-us/credentials/applied-skills/configure-secure-workloads-use-azure-virtual-networking/) assessment, confirmed by [my credential](https://learn.microsoft.com/en-us/users/sebastianczech-4206/credentials/fcae4dbc7e799f3d).

The assessment covers hands-on skills in:

* creating and configuring virtual networks (VNet)
* configuring network routing
* creating DNS zones and configuring DNS settings
* creating and configuring NSGs
* creating and configuring Azure Firewall

Applied Skills assessments are based on interactive labs – instead of answering theory questions, you perform real tasks in an Azure environment. That makes them much closer to actual work than a traditional exam.
