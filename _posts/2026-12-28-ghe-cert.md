This post outlines helpful resources for planning GitHub certification preparation - official credentials, recommended training courses, and practical reference materials.
I use this list as a compact study guide for planning and tracking my preparation.

* [GitHub credentials](https://learn.github.com/credentials)
* GitHub Copilot:
  * [Microsoft Applied Skills: Accelerate AI-assisted development by using GitHub Copilot](https://learn.microsoft.com/en-us/credentials/applied-skills/accelerate-app-development-by-using-github-copilot/)
  * [Certification - GitHub Copilot](https://learn.microsoft.com/en-us/credentials/certifications/github-copilot/?practice-assessment-type=certification)
  * [Course GH-300T00-A: GitHub Copilot](https://learn.microsoft.com/en-us/training/courses/gh-300t00)
* GitHub Advanced Security:
  * [Certification - GitHub Advanced Security](https://learn.microsoft.com/en-us/credentials/certifications/github-advanced-security)
  * [Course GH-500T00-A: GitHub Advanced Security](https://learn.microsoft.com/en-us/training/courses/gh-500t00)
* GitHub Actions:
  * [Certification - GitHub Actions](https://learn.microsoft.com/en-us/credentials/certifications/github-actions/?practice-assessment-type=certification)
  * [Course GH-200T00-A: GitHub Actions](https://learn.microsoft.com/en-us/training/courses/gh-200t00)
* Materials:
  * [Awesome Copilot](https://awesome-copilot.github.com/)
  * [Terraform Skill](https://github.com/antonbabenko/terraform-skill)

I will keep this list updated as new materials and certification updates become available.
